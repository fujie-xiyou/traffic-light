#### 8086交通灯proteus仿真

##### 运行环境
1. 仿真： Windows* + proteus7.8
2. 真实电路： 微机试验箱+自焊电路板

##### 所需器件
1. 8086(i386)
2. 8255a(可编程并行接口芯片)
3. 一位七段数码管
4. 三色led灯

##### 功能
1. 三种颜色交通灯变换
2. 东西方向和南北方向分别有一个数码管单独计时
3. 在绿灯时间只剩3秒的时候，当前方向的绿灯和倒计时会闪烁

##### 文件结构
1. traffic-light.asm
	- 8086汇编代码
2. traffic-light.dsn
	- proteus7.8 仿真电路设计文件
3. traffic-ligth.exe
	- 使用masm编译，link链接后生成的可执行文件。
